STEP 1 — INSTALL THESE PLUGINS
────────────────────────────────
✓ Elementor (free)          — page builder
✓ Elementor Pro ($59/yr)    — forms, popups, motion effects
✓ Hello Elementor (free)    — ultra-lightweight theme (made by Elementor)
✓ Rank Math SEO (free)      — SEO, local business schema
✓ WP Rocket ($49/yr)        — caching & speed  [OR free: W3 Total Cache]
✓ Smush (free)              — image compression
✓ Wordfence (free)          — security

OPTIONAL BUT USEFUL:
✓ WPForms Lite (free)       — if not using Elementor Pro forms
✓ MonsterInsights (free)    — Google Analytics connection


STEP 2 — THEME SETUP (Hello Elementor)
────────────────────────────────────────
1. Install & activate Hello Elementor theme
2. Go to Appearance > Customize > Additional CSS
3. Paste the ENTIRE contents of brand.css (in this folder)
4. Save & Publish

   That gives you:
   - Color variables
   - Google Fonts (Cormorant Garamond + Inter)
   - Base typography
   - Utility classes for buttons
